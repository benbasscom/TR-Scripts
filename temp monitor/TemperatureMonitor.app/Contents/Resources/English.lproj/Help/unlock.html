Unlocking the software

This page is not applicable to Temperature Monitor.

Hardware Monitor is electronically distributed software which is offered following a “First try, then buy” principle. You can download the application free of charge and test whether it fits your needs. During the test phase, the following limitations apply:

  • A window with the note Running in Demo Mode appears each time the application is started. It gives you a detailed recommendation whether Hardware Monitor would offer more information on the current computer than the free Temperature Monitor.
  • The application displays readings from temperature sensors only. All other types of sensors are ignored.
  • There are no other limitations regarding the functionality of the software.

If you like to use the software permanently, you'll have to place an order for the needed number of usage licenses. For each license you will receive a so-called registration code by e-mail which will allow you to switch the application from demo mode to normal operation.

Distributing or leasing the application to third parties is prohibited without prior written permission. The exact contractual obligations for licensing the software can be displayed and printed after you have opened the downloaded software package.

Ordering registration codes

Placing an order for registration codes of Hardware Monitor is possible via our distribution partner. The order can be placed via Internet, postal mail, telefax or phone. The computer on which the application should be used does not need to have an Internet link necessarily. It is only necessary that you somehow receive the e-mail message used to deliver the registration code. Delivery is possible worldwide. Payments are accepted in more than 40 different international currencies, with all common payment options supported.

Unlocking the software

The registration code needed to unlock the software will be delivered via e-mail. Please note that this code represents a value and should be archived at a safe place for future reference, e.g. by printing it on paper. The code consists of two parts, the Registration Name and the Registration Key.

It is recommended to login as an administrator when unlocking the sotware. In this case it is possible to not only unlock the application for a single user, but for all user accounts of the current computer. Perform the following steps to unlock the application for unrestricted usage:

  1. Select the menu item Hardware Monitor > Unlock Hardware Monitor…. The window MBS Product Registration will appear. It shows the current status of the registration.
  2. In this window press the button Add Key…. A sheet for entry of the registration code will appear.
  3. Transfer the registration name exactly as you have received it into the field Registration name. You can enter the data manually. However, if you currently have the registration e-mail on the same computer, it will be easier to transfer the data by the features Edit > Copy (⌘+C) and Edit > Paste (⌘+V) into the application. Please pay attention not to transfer any additional blanks or empty lines. Also note that the contents of this field is case-sensitive.
  4. With the same method transfer the registration key into the field Registration key of the application.
  5. Use the buttons at Valid for to select whether you like the registration to become active for the current user account only or for all users of this computer.
  6. Press the button Save.

If both parts of the code have been entered correctly, the window MBS Product Registration will show the successful unlock operation and the type of your license. You can close the window. If some part of the code has been entered incorrectly, an error message will be displayed and the sheet will open again to correct the problem. In this case please check both parts of the code for exact match with the e-mail which has been sent to you.

When using Mac OS X 10.7 or later, Apple no longer allows that a standard application (like Hardware Monitor) can perform operations on behalf of all users. For this reason, it won't be possible to unlock Hardware Monitor with the option Valid for: All users of this computer on such an operating system.

Deactivate the registration

You can deactivate the registration any time. Please perform the following steps:

  1. Select the menu item Hardware Monitor > Unlock Hardware Monitor….
  2. Choose the registration in the table.
  3. Press the button Remove key.
